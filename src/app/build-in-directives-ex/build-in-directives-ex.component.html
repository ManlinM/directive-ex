Structural directive: *ngFor

*ngFor is used to loop through property msgs[], and will generate a new message when button is clicked

What is happening?

msg array is created with 2 elements, and the *ngFor will loop through this array when the browser is started.

When the button is clicked, one more message is added to the msg[], and *ngFor will loop through the updated msg[] with 3 elements

Same pattern continues when the button is clicked
